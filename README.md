# Log4J Formatters

**%d**  
In yyyy-MM-dd HH:mm:ss,nnn format

**%p**   
Log level

[%t]  
For example main

**%C{1.}**  
package name

**%m%n**
Log message

**%X**  
Append MDC info

**%n**  
New line
